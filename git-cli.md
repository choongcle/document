# Git 기본
### git init
> 저장소 초기화

현재 디렉토리 내 저장소 생성
```
git init
```

<directory> 생성 + 저장소 생성
```
git init <directory>
```

최초 원격 저장소 내 저장소 생성
```
git init --bare <directory>
```
